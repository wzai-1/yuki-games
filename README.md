# yuki-games

小合集，目前只有：贪吃蛇（React + Tailwind）。

## 本地运行

```bash
npm install
npm run dev
```

## 测试

```bash
npm test
```

## 构建

```bash
npm run build
npm run preview
```

## 操作

- 键盘：方向键 / WASD
- 空格：暂停 / 继续
- 手机：在屏幕上滑动（任意区域）控制方向

## 说明

- 玩法说明默认收起（按钮可展开）
- iPhone Chrome 右边缘 1px 伪影：通过“像素对齐 + 1px 内缩”规避
